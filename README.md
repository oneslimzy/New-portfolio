# Slimzy portfolio

Open `index.html` in a browser to preview the site.

## Add your work

- In `script.js`, replace each `embedUrl: ''` with an embeddable YouTube or Vimeo URL. Clicking a motion card then plays the video in an on-site modal.
- Change motion project names, categories and colours in `motionProjects`.
- Change branding projects in `brandProjects`. The artwork areas are deliberate placeholders for your future design images.
- To use images later, replace a `.brand-art` area in `index.html` with an `<img>` and add `object-fit: cover` in `styles.css`.
